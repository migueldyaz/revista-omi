# Revista OMI Internacional 🌐

Este repositorio contiene el sitio web oficial de la **Revista OMI Internacional**, proyecto académico del Centro de Investigación de la Universidad OMI.

## 📄 Contenido

- Página principal (`index.html`)
- Secciones:
  - Acerca de la revista
  - Comité editorial
  - Directrices para autores
  - Derechos de autor
  - Artículos publicados
- Navegación institucional con diseño responsivo
- Estructura ligera y compatible con GitHub Pages

## 🚀 Visualización en línea

Este sitio está diseñado para funcionar en [GitHub Pages](https://pages.github.com/).  
Una vez publicado, será accesible en:

```
https://<tu_usuario>.github.io/<nombre_del_repositorio>/
```

## 🛠 Uso local

Puedes ver el sitio localmente con un servidor como:

```bash
python -m http.server 8000
```

Luego abre en tu navegador:  
👉 `http://localhost:8000/`
